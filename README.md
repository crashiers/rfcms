# rfcms 小程序内容发布系统

微信小程序前台源码


## 主要功能

* 推荐
* 专题
* 文章
* 签到 获得积分
* 阅读 获得积分
* 点赞 获得积分
* 组队 双倍加成
* 邀请新用户奖励 获得积分
* 积分商城 消耗积分
* 兑换卷 
* 兑换功能(引导联系微信客服兑奖)
* 评论(正在路上)
